🧩 How Do They All Fit Together?

Every Arduino program — no matter how complicated — is built from just these 4 ideas working as a team. Here's how they combine in a real robot:

// 🤖 Obstacle-Avoiding Robot — full program idea
distance = 0  ← 📦 Variable: a box to store the sensor reading
Repeat Forever:  ← 🔁 Loop: keep doing this non-stop
📡 Read sensor → store in distance  ← 📋 Sequence Step 1
IF distance < 20  ← 🔀 Condition: yes/no question
Turn motors LEFT 🔄  ← 📋 Sequence Step 2a (if YES)
ELSE  (if NO — path is clear)
Go FORWARD ➡️  ← 📋 Sequence Step 2b
↩ loop jumps back to top and runs again instantly...
📋 Sequence = the order
🔀 Condition = the choice
🔁 Loop = the heartbeat
📦 Variable = the memory
