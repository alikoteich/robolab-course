Part 2 — What is Arduino?
Arduino = Microcontroller + Easy Interface

An open-source board with an ATmega328P chip — plug in sensors, LEDs, motors and upload code from your laptop via USB. No engineering degree needed.

✅
Write code → upload via USB → runs forever, even without the computer.Power from a 9V battery and it becomes fully standalone.
⚡
All pins are 5V — never connect a 12V signal directly.The barrel jack accepts 7–12V and steps it down internally.
